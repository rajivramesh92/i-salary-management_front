# 💻 Salary Management System - Frontend (Next.js)

## 🚀 Overview

This is the frontend application for the Salary Management System. It provides a clean and interactive UI for managing employees and viewing salary insights.

---

## 🛠️ Tech Stack

* Next.js (React)
* TypeScript
* Tailwind CSS
* Recharts (Charts)
* Axios (API calls)
* Vercel (Deployment)

---

## ⚙️ Features

### 👨‍💼 Employee Management UI

* Add Employee
* View Employee List
* Edit Employee
* Delete Employee

### 🔍 Filters & Search

* Search by name
* Filter by country
* Filter by job title

### 📊 Salary Insights Dashboard

* Average salary
* Country-based insights
* Department analytics
* Visual charts using Recharts

---

## 📄 Pagination (Important)

* Implemented client-side pagination
* Displays limited records per page (default: 10)
* Optimized for handling 10,000+ employees efficiently

---

## 🔗 API Integration

* Connected with Rails backend
* Base URL configured via Axios

---

## 🚀 Deployment

* Hosted on Vercel
*https://i-salary-management-front.vercel.app

---

## ⚡ Performance Considerations

* Pagination to avoid rendering large datasets
* Optimized re-renders using React hooks
* Debounced search implementation

---

## 🧠 Engineering Decisions

* Used Next.js App Router for scalability
* Component-based architecture
* Clean separation of UI and API logic

---
